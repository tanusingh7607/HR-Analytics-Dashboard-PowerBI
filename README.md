# HR Analytics Dashboard | Power BI

## 📌 Overview

This project is an interactive HR Analytics Dashboard developed using Microsoft Power BI. It helps analyze workforce performance, employee satisfaction, engagement, work-life balance, training investment, and employee distribution through interactive visualizations and dynamic filters.

---

## 🛠️ Tools Used

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)

---

## 📊 Key Metrics

- Total Employees
- Average Employee Rating
- Average Satisfaction Score
- Average Engagement Score
- Average Work-Life Balance Score
- Training Investment (₹ Lakh)

---

## 📈 Dashboard Features

- Interactive KPI Cards
- Dynamic Slicers
- Department-wise Employee Rating Analysis
- Department-wise Satisfaction Analysis
- Department-wise Engagement Analysis
- Business Unit Analysis
- Employee Status Distribution
- Gender Distribution
- Performance Rating Distribution
- Training Cost Analysis
- Dynamic Key Insights
